# Sound Files

## نیاز به فایل صوتی

برای عملکرد صحیح برنامه، باید یک فایل صوتی با نام `notification.mp3` در این پوشه قرار دهید.

### پیشنهادات برای صدای هشدار:

1. **صدای زنگ آژیر**: برای جلب توجه فوری
2. **صدای زنگ**: صدای سنتی زنگ رستوران
3. **بیپ متوالی**: صدای بیپ‌های پشت سر هم

### منابع دانلود صدای رایگان:

- [Freesound.org](https://freesound.org/search/?q=alarm)
- [ZapSplat](https://www.zapsplat.com/sound-effect-category/alarms-and-sirens/)
- [SoundBible](https://soundbible.com/tags-alarm.html)

### مشخصات پیشنهادی:

- **فرمت**: MP3
- **نام فایل**: `notification.mp3`
- **طول**: 2-5 ثانیه
- **کیفیت**: 128kbps یا بالاتر
- **حجم صدا**: واضح و بلند

### نکته مهم:

اگر فایل صوتی را قرار ندهید، برنامه از یک URL پشتیبان استفاده می‌کند، اما برای عملکرد بهتر پیشنهاد می‌شود فایل محلی استفاده کنید.
